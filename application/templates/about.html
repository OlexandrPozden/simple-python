About

This web application was develop with purpose to learn build web applications without frameworks such as Django, Flask. Try to understand how it works underhood.

This web application has authentication and autherization features. Users can sign-up, log in, log out. Users has acces to view another published users posts. Users can create/edit/delete own posts and ask them for publshing

Admins inherit all user permissions and have ability to review and publishing users post, which was request to publish.

Not identified clients can view pages Home ('/'), Main ('/main'), Authors ('/authors'), Login ('/login'), Sign-Up ('/signup'), to access other pages client need to log in.
